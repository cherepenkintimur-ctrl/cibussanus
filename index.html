CibusSanus

Учёт продаж ресторана. Без серверов. Без головной боли.

⬇ Скачать приложение GitHub →
Что внутри
📋

Меню и категории

Управляйте блюдами и категориями. Поиск, фильтры, CRUD — всё на месте.

🛒

Заказы

Создавайте заказы за секунды. Автоматический расчёт суммы, состав, оплата.

📊

Отчёты

Выручка, средний чек, загруженность по часам, ТОП блюд — всё с фильтрами по датам.

📑

Экспорт в Excel

Одной кнопкой — полный .xlsx файл с отчётом, заказами или списком блюд.

🔄

Сортировка

Сортируйте любые списки по имени, цене, дате, сумме. Направление — одним тапом.

💾

Встроенная БД

SQLite внутри приложения. Никаких серверов — просто запусти и работай.
